# Frontend Dokümantasyonu

Frontend geliştirmesinde aşağıdaki belgeler kaynak kabul edilir:

1. [PRD](PRD.md): Ürün amacı, kullanıcılar ve frontend kapsamı
2. [MVP](MVP.md): İlk yayın kapsamı ve kabul kriterleri
3. [Görevler](TASKS.md): Uygulama sırası ve tamamlanma kontrol listesi
4. [Frontend Entegrasyonu](FRONTEND-INTEGRATION.md): Backend API, auth, hata ve SignalR sözleşmesi
5. [Claude Design Prompt](CLAUDE-DESIGN-PROMPT.md): Tasarım üretmek için kullanılacak brief

Teknik karar: Next.js App Router, TypeScript, Tailwind CSS ve shadcn/ui.

İlk uygulama listesindeki tamamlanma işaretleri yayın onayı değildir. Güncel açık işler
[MVP sertleştirme](tasks/10-mvp-hardening.md) ve [ürün evrimi](tasks/11-product-evolution.md)
dosyalarında izlenir.
