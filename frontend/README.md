# TaskManagement Frontend

TaskManagement web istemcisi burada geliştirilecektir.

## Teknik yığın

- Next.js (App Router) ve TypeScript
- Tailwind CSS
- shadcn/ui ve Lucide icons

Uygulama iskeleti, Claude Design çıktıları netleştikten sonra kurulacaktır. Ürün
kapsamı, entegrasyon sözleşmesi ve uygulama görevleri [docs](docs/README.md)
altındadır.
